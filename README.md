# UCab - MERN Stack Cab Booking Application

## Project Description

UCab is a full-stack cab booking application developed using the MERN Stack (MongoDB, Express.js, React.js, Node.js). The system allows users to register, login, view available cabs, book rides, and manage bookings. An admin panel is provided for managing cabs, users, and bookings.

## Technologies Used

* React.js
* Node.js
* Express.js
* MongoDB Atlas
* JWT Authentication
* Bootstrap
